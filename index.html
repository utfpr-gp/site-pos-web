Quem Deve Fazer Esta Pós-Graduação?

Nosso curso foi desenhado para impulsionar a carreira de diferentes perfis profissionais. Veja se você se identifica com um deles:

  • Profissionais em Transição de Carreira
    Quer migrar de outra área para o mercado de tecnologia? O curso oferece a base completa e o portfólio prático que você precisa.
  • Desenvolvedores e Profissionais de TI
    Já atua na área e quer dominar o desenvolvimento web moderno para se tornar um especialista full-stack? Aprofunde-se aqui.
  • Criativos e Entusiastas de Novas Tecnologias
    Já usa ferramentas no-code ou IA e quer ganhar autonomia para customizar e escalar suas criações? Transforme sua curiosidade em maestria.

Datas Importantes

A seleção de alunos é realizada por meio da análise de currículo. A pós-graduação é autorizada pela Resolução nº 228/25-COPPG, com 360 horas de carga horária e 120 vagas (100 para a comunidade externa e 20 para servidores da UTFPR).

FAÇA SUA PRÉ-INSCRIÇÃO

Eu sou Luciano, coordenador da Pós

Vivemos um momento fascinante na tecnologia. Com tantas ferramentas à nossa disposição, o poder de criar para a web está mais acessível do que nunca.

Mas o que realmente define um profissional de excelência não é apenas o que se cria, mas como se cria. É saber construir aplicações robustas, seguras e escaláveis, compreendendo cada etapa do processo. Esse conhecimento fundamental é o seu maior diferencial competitivo.

Nossa pós-graduação foi desenhada para ir além do superficial. Formamos desenvolvedores com visão completa, prontos para dominar o processo de ponta a ponta. Seja você um iniciante de outra área, um profissional de TI buscando se especializar, ou alguém que já usa ferramentas modernas, como a Inteligência Artificial, e quer entender a fundo o que acontece por trás do código, este curso é para você.

Estou muito feliz em liderar este projeto e te convido a fazer parte desta jornada. Vamos construir juntos o seu futuro na tecnologia?

Diferenciais da UTFPR

Descubra por que nossa pós-graduação é a escolha certa para sua carreira em desenvolvimento web.

  • Selo de Qualidade UTFPR

    Estude na primeira e única Universidade Tecnológica Federal do Brasil, reconhecida com nota máxima (5) pelo MEC. Seu certificado nasce com o peso que o mercado valoriza.

  • Foco Total na Prática e no Mercado

    Você vai construir aplicações web completas, do back-end ao front-end, e terminar o curso com um portfólio de projetos que comprova suas habilidades.

  • Comunidade Ativa e Networking de Valor

    Participe de uma comunidade vibrante no Discord, onde alunos, tutores e professores trocam código, resolvem problemas e constroem uma valiosa rede de contatos.

  • Flexibilidade do EAD, Estrutura do Presencial

    Tenha a conveniência de estudar online e, ao mesmo tempo, acesso a toda a infraestrutura da UTFPR, incluindo bibliotecas e salas de estudo 24h nos 13 campi.

  • Formação Full-Stack para Alta Empregabilidade

    O curso é desenhado para te dar uma visão 360º (full-stack), preparando você para as vagas mais cobiçadas do mercado de tecnologia.

  • Certificação Federal com Reconhecimento Nacional

    Seu investimento é recompensado com um certificado de especialista que tem validade nacional e o reconhecimento de uma universidade pública federal.

Pós em Desenvolvimento Web

Para iniciantes e profissionais de outras áreas

Mesmo sem formação em tecnologia, você pode dominar as principais ferramentas do desenvolvimento web. O curso é ideal para quem já usa IA para criar soluções, mas quer entender a fundo como funciona o processo — do código ao deploy.

  • Front-End com HTML, CSS, JavaScript e React
  • Back-End com Node.js, Express e bancos de dados
  • Git, GitHub e versionamento de código
  • Boas práticas com DevOps, Docker e deploy

Principais Tópicos do Curso

Conheça as áreas de conhecimento que você irá dominar

Fundamentos da Programação

Base sólida para quem está começando em programação ou para aqueles que já possuem alguma experiência.

Front-End

HTML5, CSS3, JavaScript, frameworks modernos (React, Vue, Angular) e design responsivo.

Back-End

Frameworks modernos para desenvolver aplicações e APIs, arquitetura de sistemas web.

DevOps

Integração contínua (CI), entrega contínua (CD), automação e orquestração com Docker.

Engenharia de Software

Metodologias ágeis, design patterns, testes automatizados e arquitetura de software.

Segurança

Criptografia, segurança em APIs, OWASP e prevenção contra ataques comuns.

Disciplinas do Curso

Desenvolvimento de Interfaces Web com HTML5 e CSS3
com Prof. Roni Banaszewski

Desenvolvimento de interfaces web responsivas:

  • Estruturação com HTML5
  • Estilização avançada com CSS3
  • Design responsivo e acessibilidade
Lógica de Programação para Desenvolvedores Web
com Prof. Renata Stange

Fundamentos de programação aplicados ao desenvolvimento web:

  • Introdução à Programação e algoritmos
  • Estruturas de controle e dados
  • Resolução de problemas com lógica
Linux para Desenvolvedores e DevOps
com Prof. Luciano Ogiboski

Fundamentos do sistema operacional Linux para o dia a dia:

  • Comandos essenciais e gerenciamento de arquivos
  • Permissões, usuários e processos
  • Configuração de ambiente de desenvolvimento
JavaScript: Desenvolvendo Aplicações Web Dinâmicas
com Prof. Roni Banaszewski

Desenvolvimento front-end interativo com JavaScript:

  • Fundamentos do JavaScript moderno (ES6+)
  • Manipulação do DOM e eventos
  • Consumo de APIs e assincronismo
Programação Orientada a Objetos
com Prof. Diego Marczal

Fundamentos da POO para código limpo e organizado:

  • Classes, objetos, herança e polimorfismo
  • Encapsulamento e abstração
  • Aplicação de padrões de projeto (Design Patterns)
Banco de Dados para Sistemas Web
com Prof. Paulo Soares

Modelagem e gerenciamento de dados para aplicações robustas:

  • Modelagem relacional e SQL avançado
  • Bancos de dados NoSQL
  • Otimização de consultas e segurança de dados
Controle de Versão e Colaboração com Git e GitHub
com Prof. Diego Marczal

Versionamento e colaboração como um profissional:

  • Fundamentos do Git e fluxo de trabalho (workflow)
  • Branches, merges e resolução de conflitos
  • Colaboração em equipe utilizando o GitHub
Gestão de Projetos Web com Metodologias Ágeis
com Prof. Renata Stange

Gerenciamento ágil para entregas de valor:

  • Princípios do Manifesto Ágil, Scrum e Kanban
  • Planejamento, execução e monitoramento de Sprints
  • Gestão de backlog e entrega contínua
Desenvolvimento Aplicações Back-End com Frameworks
com Prof. Andres Porfírio

Construção da lógica de servidor para sistemas complexos:

  • Desenvolvimento de APIs RESTful com frameworks modernos
  • Integração com bancos de dados
  • Autenticação, autorização e boas práticas
Infraestrutura Web: Servidores, Deploy e Cloud
com Prof. Andres Porfírio

Implantação e gerenciamento de aplicações na nuvem:

  • Configuração de servidores web e conteinerização (Docker)
  • Princípios de Cloud Computing (AWS, Google Cloud, Azure)
  • Processos de deploy automatizado (CI/CD)
Segurança em Aplicações Web
com Prof. Denis Silva

Proteção e segurança para aplicações web confiáveis:

  • Prevenção das principais vulnerabilidades (OWASP Top 10)
  • Testes de segurança e análise de código
  • Conformidade com a LGPD e boas práticas
Inovação e Tendências em Desenvolvimento Web
com Prof. Luciano Ogiboski

Exploração do futuro do desenvolvimento web:

  • Análise de tecnologias emergentes e novos paradigmas
  • Relatos de profissionais sobre suas experiências reais
  • Discussões sobre os desafios e oportunidades da área

Nossos Professores

Conheça os professores que fazem parte do nosso programa de especialização

Professor Doutor

Andres Porfírio

Minha paixão está nos bastidores da web, na 'sala de máquinas' onde a mágica realmente acontece. Vamos mergulhar fundo no desenvolvimento back-end, utilizando frameworks modernos para construir as APIs e a lógica que dão vida às aplicações. Além disso, vamos desmistificar a infraestrutura, aprendendo a colocar nossos projetos no ar em servidores na nuvem, garantindo performance e escalabilidade.

Professor Mestre

Denis Silva

Em um mundo onde tudo está conectado, a segurança não é um opcional, é a fundação de um bom software. Meu foco é trazer uma mentalidade de 'defesa' para o seu desenvolvimento. Vamos explorar as principais vulnerabilidades em aplicações web e, mais importante, aprender as técnicas e boas práticas para construir sistemas seguros desde o primeiro dia, protegendo os dados dos usuários e a integridade do seu projeto.

Professor Doutor

Diego Marczal

Acredito que escrever um bom código vai além da sintaxe; é sobre organização e colaboração. Comigo, vocês vão dominar a Programação Orientada a Objetos para criar sistemas mais limpos e reutilizáveis, e usaremos ferramentas como Git e GitHub para aprender a trabalhar em equipe, que é como os melhores softwares do mundo são construídos hoje.

Professor Doutor

Luciano Ogiboski

A minha visão para esta pós é conectar vocês com o que há de mais atual e relevante no universo da tecnologia. Vamos explorar juntos as tendências que estão moldando o futuro do desenvolvimento, ao mesmo tempo em que construímos uma base sólida sobre a infraestrutura (Linux, DevOps) que sustenta todas as grandes aplicações. Meu objetivo é formar profissionais preparados não apenas para o hoje, mas para o amanhã.

Professor Doutor

Paulo Soares

Toda grande aplicação web é movida por dados. Minha especialidade é justamente essa: como modelar, armazenar e acessar informações de forma eficiente e segura. Comigo, vocês vão aprender a projetar e gerenciar bancos de dados, uma competência absolutamente crítica e valorizada no mercado, garantindo que seus sistemas sejam não apenas bonitos, mas também inteligentes e robustos.

Professora Doutora

Renata Stange

Para mim, um projeto web de sucesso se apoia em dois pilares: um código com lógica impecável e um processo de gestão que funciona. Nas minhas aulas, vamos trabalhar nessas duas frentes: construiremos o raciocínio lógico essencial para programar e aprenderemos a gerenciar projetos com metodologias ágeis, garantindo que suas ótimas ideias saiam do papel e se transformem em entregas de valor.

Professor Doutor

Roni Banaszewski

Sou fascinado pela experiência do usuário e pela forma como uma interface bem construída pode encantar e engajar. Nas minhas aulas, vamos transformar o design em realidade, estruturando páginas com HTML5, estilizando com CSS3 e, o mais importante, adicionando vida e interatividade com JavaScript. Meu objetivo é que você se torne mestre na arte de criar o lado visível e dinâmico da web.
